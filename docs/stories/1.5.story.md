## User Story

**Como** criador de conteúdo/product manager que trabalha com Claude Code,
**Quero** um guia prático que explique como criar brand skills para gerar documentos (PDFs, apresentações, landing pages) que respeitam a identidade visual da marca,
**Para que** eu consiga automatizar a geração de conteúdo on-brand sem precisar editar manualmente o design em cada documento.

---

## Acceptance Criteria

- [ ] AC1: O guia explica o que é uma brand skill no Claude Code (propósito, casos de uso).
- [ ] AC2: Tutorial passo-a-passo mostra como criar uma brand skill em menos de 10 minutos.
- [ ] AC3: Demonstra 3+ casos de uso práticos (PDF, apresentação, landing page).
- [ ] AC4: O conteúdo é estruturado como HTML estático, em light mode, seguindo o design system (sinapse.space + Apple).
- [ ] AC5: CTA duplo: sutil no topo ("Explore brand skills no sinapse.space") + botão forte no final com link para sinapse.space.
- [ ] AC6: Inline styles forçados para garantir contraste do botão CTA em light mode.

---

## Scope

### In

- Nova página HTML em `guias/claude-code-brand-skills.html`
- Seções: Hero, O que são Brand Skills, Tutorial (passo-a-passo), Casos de Uso (3 exemplos), CTA final
- Estilos integrados ao CSS compartilhado ou inline
- One-shot workflow (sem refinamento iterativo)

### Out

- Integração com API do sinapse.space
- Geração automática de exemplos
- Customização avançada de brand skills (fora do escopo inicial)
- Criação de brand skills de verdade no Claude Code durante o guia

---

## Dependencies

- Documentação oficial do Claude Code sobre brand skills (caso exista)
- Design system confirmado: light mode, sinapse.space colors, system fonts (Inter/SF Pro)
- URL de destino: sinapse.space (ou padrão CTA do projeto)
