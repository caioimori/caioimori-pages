O que o Computer Use faz

Quando Claude Code não tem um connector, plugin ou outra integração mais precisa para resolver a tarefa, ele pode interagir diretamente com a sua tela: abrir janelas, navegar no browser, clicar, rolar, digitar e acionar ferramentas locais.

Como ele decide agir

Segundo a documentação oficial, Claude tenta usar a rota mais precisa primeiro. Na prática, a ordem é:

Connectors e integrações Browser quando fizer sentido Interação direta com a tela

O que isso habilita

  • Abrir apps e arquivos locais sem você pegar no mouse.
  • Navegar em interfaces que não têm MCP ou connector dedicado.
  • Usar browser, IDE e ferramentas de dev no mesmo fluxo.
  • Continuar uma tarefa no seu computador enquanto você está longe dele.
01

Pré-requisitos reais

A documentação pública da Claude converge nesses pontos. Se você quer só validar elegibilidade, essa é a lista curta.

1. Claude Desktop instalado

Baixe ou atualize em claude.com/download.

2. macOS suportado

O app desktop exige macOS 11 ou superior. O Computer Use no Claude Code está, por enquanto, restrito ao Mac.

3. Plano pago correto

Computer Use no Claude Code está liberado em research preview para assinantes Pro e Max.

Resumo rápido: app atualizado, plano Pro ou Max, desktop acordado, Claude Desktop aberto. Sem isso, não tem Computer Use funcionando.

02

Passo a passo para ativar

Esse é o fluxo documentado hoje pela Anthropic para ligar o recurso dentro do Claude Desktop e começar a usar no Claude Code.

  1. Atualize o Claude Desktop

    Baixe a versão mais recente em claude.com/download, abra o app e faça login na sua conta.

  2. Abra as configurações do desktop

    No Claude Desktop, entre em Settings > General. O toggle do Computer Use fica nessa área, sob as configurações do app desktop.

  3. Ligue o toggle de Computer Use

    Ative a chave Computer use. Se ela não estiver visível, normalmente o problema é plano incompatível, app desatualizado ou rollout ainda não liberado para a sua conta.

  4. Abra uma sessão do Claude Code no desktop

    Entre no Claude Code dentro do app desktop e comece uma sessão normal. O Computer Use só entra em cena quando a tarefa realmente exige interagir com a sua máquina.

  5. Peça uma tarefa que exija tela ou app local

    Exemplos: abrir um app, navegar num dashboard interno, mexer numa planilha local, rodar uma ferramenta visual ou operar algo no browser sem integração pronta.

  6. Autorize o acesso ao primeiro app

    Claude vai pedir permissão antes de tocar cada aplicativo novo. Aprove ou bloqueie caso a caso. Você também pode interromper a execução a qualquer momento.

Dica prática: comece com tarefas simples e bem delimitadas. A própria Anthropic avisa que o recurso ainda pode errar em fluxos longos ou interfaces mais complexas.

03

Como pedir do jeito certo

O ganho aqui não é só ligar o toggle. É formular pedidos que façam o Claude operar de forma previsível e segura.

Prompts que funcionam melhor

  • "Abra o app X, exporte o relatório Y e salve na pasta Z."
  • "Entre no dashboard interno, filtre por março e me traga os números principais."
  • "Abra meu projeto, rode os testes e me diga onde quebrou."
  • "Navegue nessa tela e complete só os campos A, B e C."

Evite começar assim

  • "Resolve tudo aí."
  • "Mexe no que precisar e vê no que dá."
  • Pedidos sem app alvo, resultado esperado ou limite claro de ação.
  • Tarefas sensíveis envolvendo finanças, saúde, dados legais ou contas críticas.
04

Permissões e segurança

O Computer Use toca sua tela real, não um navegador isolado. Então a regra é simples: trate o recurso como algo poderoso e não como uma automação inocente.

Faça isso

  • Comece pelos apps em que você realmente confia.
  • Feche janelas com documentos sensíveis antes de usar.
  • Conceda acesso por aplicativo com bastante critério.
  • Use prompts específicos para reduzir interpretações erradas.
  • Saiba como interromper ou revogar acesso rapidamente.

Evite isso

  • Apps bancários, saúde, governo ou qualquer sistema altamente sensível.
  • Trabalhar com dados pessoais de terceiros visíveis na tela.
  • Delegar operações irreversíveis sem supervisão.
  • Assumir que as salvaguardas são perfeitas.
  • Dar acesso amplo quando um fluxo mais simples resolveria.
05

Limites atuais

A Anthropic ainda trata o recurso como prévia de pesquisa, então vale ajustar expectativa para não usar Computer Use no tipo errado de trabalho.

Fluxos longos ou complexos às vezes precisam de uma segunda tentativa.

É mais lento que integração direta

Se houver connector, MCP ou plugin, esse caminho tende a ser melhor.

06

Opcional: usar com Dispatch

Se a sua conta já tiver acesso ao Dispatch, você pode disparar tarefas pelo celular e deixar o desktop trabalhar no seu lugar. Isso não é obrigatório para ativar o Computer Use, mas combina muito bem com ele. Para esse fluxo, você precisa do app mobile da Claude atualizado e conexão ativa nos dois dispositivos.

O que já está documentado

A central de ajuda explica que, no fluxo contínuo do Dispatch, tarefas de desenvolvimento podem ser encaminhadas para sessões do Claude Code, enquanto o resultado volta para a mesma thread.

Como preparar o fluxo

O onboarding detalhado está documentado na área do Cowork: abra Cowork no celular ou no desktop, entre em Dispatch, toque em Get started, libere acesso a arquivos e a opção de manter o computador acordado, e finalize a configuração.
