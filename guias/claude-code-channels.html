Claude Code Channels:
Telegram & Discord

Conecte o Claude Code ao seu Telegram ou Discord e converse com a IA direto do celular — enquanto ela trabalha nos seus arquivos reais.

Este guia é um conteúdo SINAPSE — aprenda IA aplicada com clareza e execução →

O que são Channels?

Channels são uma forma de empurrar mensagens para dentro de uma sessão do Claude Code que já está rodando no seu computador.

Na prática: você abre o Claude Code no terminal, conecta um canal (Telegram ou Discord) e passa a conversar com o Claude de qualquer lugar — do celular, do trabalho, de onde quiser. A IA lê sua mensagem, executa o que for preciso nos seus arquivos locais e responde direto na conversa.

Por que isso importa

Diferente de usar o Claude na web, aqui a IA tem acesso ao seu projeto real. Ela lê, edita e roda comandos no seu código — e você acompanha tudo pelo chat.

Antes de começar

Confira se você tem tudo pronto:

  1. Claude Code v2.1.80+ Abra o terminal e rode claude --version. Se estiver abaixo, atualize com npm install -g @anthropic-ai/claude-code@latest
  2. Login via claude.ai Channels só funcionam com login pelo claude.ai. API keys e Console não são suportados.
  3. Bun instalado Os plugins de canal rodam com Bun. Rode bun --version para verificar. Se não tiver, instale em bun.sh
  4. Plano Pro ou Max Se você usa plano pessoal (Pro/Max), channels já estão disponíveis. Planos Team/Enterprise precisam de liberação do admin.
Telegram

Setup no Telegram

Você vai criar um bot no Telegram e conectá-lo ao Claude Code. O bot é seu — só você vai usar.

  1. Crie o bot no BotFather Abra o Telegram e procure por @BotFather. Envie o comando /newbot. Escolha um nome de exibição e um username que termine com bot (ex: meuclaude_bot). O BotFather vai devolver um token — copie ele.
  2. Instale o plugin no Claude Code No terminal com o Claude Code aberto, rode:
    /plugin install telegram@claude-plugins-official

    Depois rode /reload-plugins para ativar o comando de configuração.

  3. Configure o token do bot Rode o comando abaixo substituindo pelo seu token:
    /telegram:configure SEU_TOKEN_AQUI

    Isso salva o token em ~/.claude/channels/telegram/.env

  4. Reinicie com channels ativado Saia do Claude Code e abra de novo com a flag --channels:
    claude --channels plugin:telegram@claude-plugins-official
  5. Pareie sua conta Vá no Telegram e mande qualquer mensagem pro seu bot. Ele vai responder com um código de pareamento. Volte pro Claude Code e rode:
    /telegram:access pair CÓDIGO
  6. Bloqueie o acesso Para que só você possa enviar mensagens pro bot:
    /telegram:access policy allowlist
Pronto

Agora é só mandar mensagem pro bot no Telegram. O Claude lê, executa no seu projeto e responde na conversa.

Discord

Setup no Discord

No Discord, você cria um bot pelo Developer Portal, adiciona num servidor e conecta ao Claude Code.

  1. Crie o bot no Developer Portal Acesse o Discord Developer Portal. Clique em New Application e dê um nome. Na seção Bot, crie um username, clique em Reset Token e copie o token.
  2. Ative o Message Content Intent Ainda nas configurações do bot, vá em Privileged Gateway Intents e ative o Message Content Intent. Sem isso, o bot não consegue ler suas mensagens.
  3. Convide o bot pro servidor Vá em OAuth2 → URL Generator. Marque o scope bot e ative estas permissões:
    • View Channels
    • Send Messages
    • Send Messages in Threads
    • Read Message History
    • Attach Files
    • Add Reactions

    Abra a URL gerada e adicione o bot ao seu servidor.

  4. Instale o plugin no Claude Code No terminal com o Claude Code aberto, rode:
    /plugin install discord@claude-plugins-official

    Depois rode /reload-plugins para ativar o comando de configuração.

  5. Configure o token do bot Rode o comando abaixo com o token que você copiou:
    /discord:configure SEU_TOKEN_AQUI
  6. Reinicie com channels ativado Saia do Claude Code e abra de novo:
    claude --channels plugin:discord@claude-plugins-official
  7. Pareie sua conta Mande uma DM pro bot no Discord. Ele responde com um código de pareamento. No Claude Code, rode:
    /discord:access pair CÓDIGO
  8. Bloqueie o acesso Para garantir que só você usa o bot:
    /discord:access policy allowlist
Pronto

Mande uma DM pro bot no Discord e o Claude vai responder enquanto trabalha nos seus arquivos locais.

Se algo der errado

"Plugin not found"

Rode estes dois comandos e tente instalar de novo:

/plugin marketplace add anthropics/claude-plugins-official
/plugin marketplace update claude-plugins-official

Bot não responde

O bot só funciona enquanto o Claude Code está rodando com --channels. Confirme que você reiniciou com a flag correta.

Bun não encontrado

Os plugins precisam do Bun. Instale com:

curl -fsSL https://bun.sh/install | bash

Plano Team/Enterprise

O admin precisa ativar channels em claude.ai → Admin settings → Claude Code → Channels antes de qualquer membro usar.

Dicas de uso

  • Sessão precisa estar aberta. Channels não rodam em background por padrão. Mantenha o terminal com Claude Code aberto — ou rode em um terminal persistente (tmux, screen).
  • Múltiplos canais ao mesmo tempo. Você pode conectar Telegram e Discord na mesma sessão separando por espaço: claude --channels plugin:telegram@claude-plugins-official plugin:discord@claude-plugins-official
  • Respostas aparecem no chat, não no terminal. Quando o Claude responde pelo channel, o terminal mostra o tool call e "sent" — a resposta real aparece no Telegram/Discord.
  • Permissões no terminal. Se o Claude precisar de aprovação para alguma ação e você não estiver no terminal, a sessão pausa até você voltar.
