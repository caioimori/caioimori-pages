Claude Auto Mode

Deixe o Claude Code trabalhar sozinho enquanto você faz outra coisa — sem precisar aprovar cada ação manualmente.

O que é o Auto Mode?

Quando você usa o Claude Code no modo padrão, ele pede permissão para cada ação — escrever um arquivo, rodar um comando, criar uma pasta. Seguro, mas interrompe o fluxo o tempo todo.

O Auto Mode é um meio-termo inteligente. Ele deixa o Claude tomar decisões automaticamente sobre o que é seguro, bloqueia o que parece perigoso e só te chama quando realmente precisa.

Na prática

Você manda o Claude fazer algo grande — tipo refatorar um módulo inteiro — e vai tomar um café. Ele faz o trabalho sozinho, sem te pedir confirmação a cada arquivo salvo.

Como funciona por baixo

Antes de cada ação, um classificador automático analisa o que o Claude quer fazer e decide:

Ação segura

Escrever arquivo, rodar lint, criar pasta — executa direto, sem interromper.

Ação arriscada

Deletar em massa, acessar dados sensíveis — bloqueia e sugere uma alternativa mais segura.

Risco persistente

Se o Claude insiste em algo bloqueado, o sistema te chama com um prompt manual.

Resultado

Menos interrupções, sem abrir mão da segurança. O melhor dos dois mundos.

Auto Mode vs. outros modos

Modo Comportamento Quando usar
Padrão Pede permissão para tudo Tarefas sensíveis, aprendizado
Auto Mode Classifica e decide sozinho Tarefas longas, refatoração, builds
Skip Permissions Aprova tudo, sem filtro Ambientes isolados (containers, VMs)
Importante

O Auto Mode reduz risco, mas não elimina totalmente. A Anthropic recomenda usar preferencialmente em ambientes isolados para tarefas críticas.

Como ativar

Três formas dependendo de onde você usa o Claude Code:

  1. Terminal (CLI) Rode o Claude Code com a flag:
    claude --enable-auto-mode

    Ou, dentro de uma sessão ativa, use Shift+Tab para alternar entre os modos de permissão até chegar no Auto Mode.

  2. VS Code Vá em Settings → Claude Code e ative o Auto Mode. Depois, na sessão aberta, selecione o modo no dropdown de permissões.
  3. Desktop App Vá em Settings → Claude Code, ative o toggle do Auto Mode. Selecione na sessão pelo dropdown.

O que o classificador bloqueia

O sistema analisa cada ação antes de executar. Estes são os tipos de operação que ele barra automaticamente:

  • Deleção em massa — remover múltiplos arquivos ou diretórios inteiros
  • Exfiltração de dados — tentativas de enviar dados sensíveis para fora
  • Código malicioso — execução de scripts suspeitos ou destrutivos
  • Operações irreversíveis — ações que não podem ser desfeitas facilmente

Quando algo é bloqueado, o Claude não trava — ele recebe orientação para encontrar um caminho alternativo e mais seguro.

Modelos

Dicas de uso

  • Comece com tarefas médias. Teste numa refatoração de um módulo antes de soltar em projetos inteiros.
  • Git é seu amigo. Faça commit antes de soltar o Auto Mode. Se algo der errado, você volta fácil.
  • Use em ambientes isolados para tarefas pesadas. Container ou VM dão uma camada extra de segurança.
  • Pode consumir um pouco mais de tokens. O classificador roda antes de cada ação — impacto pequeno em custo e latência.
  • Combine com Channels. Ative Auto Mode + Telegram/Discord e acompanhe o progresso pelo celular.
